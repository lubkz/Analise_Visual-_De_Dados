# Análise de Dados: Performance de Jogos por Gênero
Este projeto realiza uma análise exploratória de dados (EDA) em um conjunto de dados de jogos (formato CSV), utilizando Python e a biblioteca Pandas para extrair insights sobre avaliações e popularidade.

## 🔍 O que foi analisado:
Média de Avaliações por Gênero: Comparativo entre análise bruta e análise filtrada (removendo gêneros com menos de 10 aparições para evitar distorções estatísticas).

Proporcionalidade de Engajamento: Identificação de quais gêneros possuem a melhor média de avaliações positivas em relação ao volume total de votos.

Rankings de Performance: Levantamento dos 5 jogos com melhor e pior avaliação na base de dados.

## 🛠️ Tecnologias Utilizadas:
Python: Lógica e manipulação de arquivos.

Pandas: Limpeza, filtragem e agregação de dados.

Jupyter Notebook: Documentação visual e exibição de gráficos.
